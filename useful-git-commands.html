Compare just file status changes between any two branches
git diff --name-status firstBranch..secondBranch
Pickup upstream changes, but ignore local changes. useful for having templates checked in that you then modify locally (e.g. example local dev application settings) but don't want to check back in.
git update-index --skip-worktree
Speed up Git's behavior by ignoring unnecessary files that should never be changed locally. Useful in very large projects.
git update-index --assume-unchanged
